public static function formatNumber(aValue:*, aMaxDecimals:int = 2, aForceDecimals:Boolean = true, aSiStyle:Boolean = false):String

Корректное форматирование дробных чисел в строку.

Parameters

aValue:* — Значение которое необходимо перевести в строку.
 
aMaxDecimals:int (default = 2) — Максимально допустимое количество цифр в дробной части.
 
aForceDecimals:Boolean (default = true) — Определяет принудительное добавление дробной части даже если значение не является дробным.
 
aSiStyle:Boolean (default = false) — Определяет стиль отображения, если true то в качестве разделителя используется точка, иначе запятая.

Returns
String — Возвращает текстовое отформатированное значение.
formatString()method 
public static function formatString(aFormat:String, ... args):String

Форматирует строку в .Net стиль, заменяет значение в фигурных скобках на переданные аргументы.

Пример использования:

         var str:String = AntFormat.formatString("Hello {0}! It's a {1}.", "World", "UFO");
         trace(str); - // Output the "Hello World! It's a UFO.";

Parameters

aFormat:String — Текст который будет отформатирован.
 
... args — Аргументы через запятую которые будут добавлены в форматируемую строку.

Returns
String — Возвращает отформатированную строку.
